설정

양손 모드: 왼쪽 Shift(또는 F)와 오른쪽 Shift(또는 J) 키를 동시에 누르고 있어야 작동합니다.
스페이스바 단독: 스페이스바 하나를 길게 누르고 있다가 떼면 타이머가 시작됩니다.

키보드 단축키 안내

  • L-Shift / F : 왼쪽 패드 터치
  • R-Shift / J : 오른쪽 패드 터치
  • Spacebar : 스페이스바 단독 모드일 때 터치 / 리셋 (타이머 정지 상태)
  • Esc : 설정 닫기 / 강제 정지
